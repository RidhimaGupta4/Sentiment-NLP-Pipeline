# outputs/

This folder contains all static charts generated by `scripts/03_eda_charts.py`.

| File | Description |
|---|---|
| `01_sentiment_distribution.png` | Bar and donut chart of positive, negative, neutral label counts |
| `02_sentiment_by_category.png` | Grouped bar chart — sentiment split across 6 product categories |
| `03_topic_distribution.png` | Horizontal bar chart — review volume by topic |
| `04_monthly_trend.png` | Line chart — % positive vs % negative reviews per month 2022–2024 |
| `05_priority_breakdown.png` | Priority complaints by category and by signal type |
| `06_company_sentiment_heatmap.png` | Heatmap — % positive reviews per company per category |
| `07_confusion_matrix.png` | DistilBERT confusion matrix on held-out test set |

## How to regenerate all charts

```bash
python scripts/03_eda_charts.py
```
