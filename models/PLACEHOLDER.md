# models/

This folder contains all model artefacts, evaluation results, and configuration files.

| File or Folder | Description |
|---|---|
| `distilbert_sentiment/` | HuggingFace model folder — config, tokeniser, and training args |
| `eval_results.json` | Full confusion matrix, per-class F1, overfitting check, leakage check |
| `model_card.json` | Complete model card — architecture, training config, limitations, intended use |
| `training_results.json` | Training run summary auto-generated by demo script |

## Model performance summary

| Metric | Score |
|---|---|
| Test Accuracy | 87.6% |
| F1 Macro | 87.1% |
| F1 Negative | 90.8% |
| F1 Neutral | 72.2% |
| F1 Positive | 91.1% |
| Avg Inference | ~42ms |

## How to regenerate all artefacts

```bash
# Demo mode — no GPU needed
python scripts/02_train_model.py --demo

# Full training — GPU recommended
python scripts/02_train_model.py --epochs 3 --batch_size 16 --lr 2e-5
```
