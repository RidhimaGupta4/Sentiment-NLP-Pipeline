# data/processed/

All processed datasets generated by `scripts/01_generate_data.py`.

| File | Rows | Description |
|---|---|---|
| `reviews.csv` | 2,000 | Master labelled review dataset — ground truth only |
| `company_summary.csv` | 30 | Aggregated sentiment metrics per company |
| `monthly_trend.csv` | 36 | Monthly sentiment trends January 2022 to December 2024 |
| `topic_distribution.csv` | 9 | Review counts and percentages per topic |
| `priority_complaints.csv` | 596 | Reviews flagged with safety, legal, or fraud signals |
| `tfidf_keywords.json` | — | TF-IDF extracted keywords per topic cluster (visualisation only) |

## Schema — reviews.csv

| Column | Type | Description |
|---|---|---|
| `review_id` | string | Unique identifier e.g. REV00001 |
| `date` | date | Review date YYYY-MM-DD |
| `year` | int | 2022, 2023, or 2024 |
| `month` | int | 1 to 12 |
| `quarter` | string | Q1, Q2, Q3, or Q4 |
| `company` | string | One of 30 UK companies |
| `category` | string | One of 6 product categories |
| `rating` | int | Star rating 1 to 5 |
| `review_text` | string | Full review text |
| `true_sentiment` | string | Ground-truth label — positive, neutral, or negative |
| `primary_topic` | string | Rule-based topic label |
| `all_topics` | string | All detected topics pipe-separated |
| `is_priority` | bool | True if safety, legal, or fraud signals detected |
| `priority_signals` | string | Signal categories detected pipe-separated |
| `word_count` | int | Number of words in review |
| `char_count` | int | Number of characters in review |
